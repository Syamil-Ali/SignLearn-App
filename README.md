# SignLearn

SignLearn is a Vite + React app for school students learning sign language.
It is a static frontend app and does not require any server-side API keys.

## Run locally

1. Install dependencies with `npm install`
2. Start the development server with `npm run dev`
3. Build for production with `npm run build`

## Production notes

- Student progress is stored only in the browser on that device.
- Use the in-app reset button on shared school devices.
- Replace placeholder or missing media with real school-approved sign content before wider rollout.
